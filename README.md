# Mood Atlas

一个基于静态前端实现的 64 型人格测试网页。

当前版本：

- `0.1.0-beta.12`

在线地址：

- https://lengzhe1984.github.io/mood-atlas/

仓库地址：

- https://github.com/Lengzhe1984/mood-atlas

## Features

- 48 道量表题
- 6 个隐藏维度计分
- 64 个独立结果映射
- 首页、整页式答题流程、结果页
- GitHub Pages 自动部署

## Version Policy

- 当前项目按 `beta` 版本维护。
- 每次发布更新时，同步更新 `package.json` 版本号和本 README。
- 版本说明优先记录用户可感知的变化，比如 UI、结果文案、代号、部署方式。

## Release Notes

### `0.1.0-beta.1`

- 初始 beta 版上线。
- 接入 48 道题、6 个隐藏维度和 64 个结果映射。
- 完成基础首页、答题流程、结果页与结果库。

### `0.1.0-beta.2`

- 结果页升级为完整格式。
- 每个结果补齐一句话结论、吐槽描述、优点、隐藏痛点、高频弹幕。
- 接入 GitHub Pages 自动部署，网站正式可公开访问。

### `0.1.0-beta.3`

- 更新结果代号与英文名展示。
- 同步调整部分中文名：
  `回弹人`、`氛围精`、`看破人`、`记账人`、`回春人` 等。
- 搜索、结果卡、结果页、分享文案全部支持 `代号 / English / 中文名`。

### `0.1.0-beta.4`

- 重做答题页布局，改为整页式分组答题，尽量减少翻页。

- 优化桌面端浏览效率，增加分组导航和右侧进度摘要。

### `0.1.0-beta.5`

- 首页保留单个结果气质预览和答题入口，让手机端、桌面端进入测试都更直接。
- 同步维护版本号到 `package.json`、`package-lock.json`、应用界面和本 README。
